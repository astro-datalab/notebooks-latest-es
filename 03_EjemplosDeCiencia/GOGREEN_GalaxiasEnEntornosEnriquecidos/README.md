# Descripción breve

## 0_GOGREENDr2ServicioDeArchivos.ipynb

Este notebook utiliza el servicio de archivos para mostrar los archivos disponibles como parte del conjunto de datos GOGREEN DR1, el cual incluye datos de GOGREEN y GCLASS.

## 1_GOGREENDr2AccesoADatosEnDataLab.ipynb

Este notebook está adaptado de la versión original producida por el equipo del survey GOGREEN. Esta versión carga los catálogos mediante un servicio TAP (usando consultas SQL simples) y accede a todos los archivos de datos necesarios (archivos FITS con espectros 1-d y 2-d, imágenes, etc.) mediante el servicio de archivos de Data Lab para GOGREEN DR1.

## 2_GOGREENDr2SIA.ipynb

Este notebook demuestra cómo las imágenes de GOGREEN DR1 pueden consultarse en Data Lab utilizando el servicio Simple Image Access (SIA), y cómo recuperar cutouts alrededor de posiciones en el cielo.

## 3_GOGREENDr2MembresiaACumulo.ipynb

Este notebook tiene por objetivo visualizar las posiciones en el cielo y los corrimientos al rojo (*redshifts*) de galaxias en función de su pertenencia al cúmulo.

## 4_GOGREENDr2Spec1DCorrimientoAlRojo.ipynb

Este notebook ilustra características de líneas de emisión en espectros 1D y 2D y muestra cómo derivar el corrimiento al rojo (*redshift*) y el ancho equivalente del doblete \[O II\]3727 a partir de un ajuste Gaussiano.
